Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

В test используется именованный возвращаемый параметр, а в anotherTest - нет. В этом все отличие, потому что в функции test фнонимная функция в defer видит именнованный параметр и может повлиять на возвращаемое значение, а в anotherTest - нет, так как сначала посчиталось возвращаемое значение, а потом выполнилась отложенная функция (если нет именнованных возвращаемых параметров, как в случае функции test).
Defer'ы выполняются перед завершением функции в обратном порядке (Складываюся в стек (LIFO)). Кроме случаев os.Exit, тогда defer'ы не выполняются.
```
